# 快速开始 - 预处理预览功能

## 快速测试新功能

### 步骤1：创建测试数据

运行测试脚本创建测试Excel文件：

```bash
python test_preview_function.py
```

这会生成 `测试数据_预览功能.xlsx`，包含5行带有摘要的测试数据。

### 步骤2：启动程序

```bash
python 亿看智能识别系统.py
```

### 步骤3：导入测试数据

1. 在"Excel凭证转换"标签页
2. 点击"选择原始Excel"，选择 `测试数据_预览功能.xlsx`
3. 确认"启用摘要智能识别（自动填充字段）"已勾选 ✓
4. 点击"自动识别匹配"
5. 确认"摘要"列已映射到摘要字段

### 步骤4：查看预处理预览

1. 点击"开始转换并导出"按钮
2. **预处理预览窗口会自动弹出**

### 步骤5：在预览窗口中

#### 查看"识别详情"页：

你会看到5行数据的识别结果，例如：

```
┌─ 第 1 行 ──────────────────────────────────────┐
│ 摘要: 销售鞋类商品给ABEL CAPOTE，金额500美元     │
│ 识别结果:                                       │
│   • 摘要编码: 01                                 │
│   • 类型: 1                                      │
│   • 科目编码: 1122                               │
│   • 往来单位名: ABEL CAPOTE                      │
│   • 金额: 500.0                                  │
└────────────────────────────────────────────────┘
```

#### 切换到"转换结果预览"页：

查看最终转换后的表格数据，包含所有13个字段列。

### 步骤6：确认导出

1. 如果识别结果正确，点击"确认并导出"
2. 如果需要调整，点击"取消"返回主界面

## 功能特点

### ✅ 智能识别
- 自动从摘要中提取业务类型、往来单位、金额等信息
- 支持11种业务类型（销售、采购、收付款、费用、库存等）
- 支持多种金额格式（元、美元、USD、千位分隔符等）

### ✅ 预处理预览
- 导出前查看所有识别结果
- 两个视图：识别详情 + 转换结果
- 可随时取消并调整

### ✅ 优先级控制
- 手动映射 > 智能识别 > 空值
- 灵活组合使用

## 注意事项

1. **首次使用前确保基础数据已导入**
   - 程序启动时会自动检查并导入
   - 或通过菜单：基础数据 → 导入基础数据

2. **摘要列必须映射**
   - 智能识别需要"摘要"字段
   - 使用"自动识别匹配"或手动选择

3. **预览窗口是模态的**
   - 打开时主窗口锁定
   - 必须选择"确认"或"取消"才能继续

4. **取消不会保存数据**
   - 可以安全返回调整
   - 重新转换会重新识别

## 故障排除

### 问题1：预览窗口没有弹出
**检查**：
- "启用摘要智能识别"是否勾选？
- "摘要"列是否已映射？
- 摘要内容是否包含可识别关键词？

### 问题2：识别结果不准确
**解决**：
- 点击"取消"返回
- 手动映射对应字段（会覆盖智能识别）
- 或修改摘要内容使其更明确
- 重新执行转换

### 问题3：窗口太小看不清
**解决**：
- 拖拽窗口边缘调整大小
- 使用滚动条查看完整内容
- 建议在1920x1080或更大屏幕使用

# 基础数据编辑功能说明

## 功能概述

亿看智能识别系统现已集成完整的基础数据管理功能，支持对7类基础数据进行增删改查操作。

## 支持的数据类型

| 数据类型 | 表名 | 记录数 | 主要字段 |
|---------|------|--------|---------|
| 币种 | currency | 1 | 编码、名称、汇率、使用类型 |
| 部门 | department | 2 | 编码、名称、是否启用 |
| 仓库 | warehouse | 3 | 编码、名称、类型、生产流程、外包单位 |
| 科目编码 | account_subject | 254 | 科目名、借贷类型、科目类型、上级科目 |
| 品目信息 | product | 1674 | 编码、名称、规格、单位、价格、尺寸 |
| 往来单位 | business_partner | 480 | 编码、名称、联系人、电话、邮箱、分类 |
| 账户 | bank_account | 21 | 编码、名称、会计科目、外币存折 |

## 操作指南

### 1. 打开查询窗口

通过主菜单访问：

```
基础数据 → 查询科目编码/往来单位/品目信息/部门/账户
```

### 2. 新增记录

**步骤**：
1. 点击查询窗口右上角的 **"新增"** 按钮
2. 在弹出的编辑对话框中填写各字段
3. 必填字段：
   - **编码 (code)** - 必须唯一，不能与现有记录重复
   - **名称 (name)** - 记录的名称
4. 数字字段会自动转换类型（如：汇率、价格、尺寸等）
5. 空白字段将保存为 NULL
6. 点击 **"保存"** 完成添加
7. 点击 **"取消"** 放弃操作

**示例 - 添加新部门**：
```
编码: TEST001
名称: 测试部门
是否启用: 是
```

### 3. 编辑记录

**步骤**：
1. 在列表中 **单击选中** 要编辑的记录
2. 点击 **"编辑"** 按钮
3. 在编辑对话框中修改字段值
4. **ID字段为只读**，不可修改
5. 修改完成后点击 **"保存"**
6. 列表会自动刷新显示更新后的数据

**注意**：
- 编辑时也要保证编码的唯一性
- 如果修改编码与其他记录冲突，会提示错误

### 4. 删除记录

**步骤**：
1. 在列表中 **单击选中** 要删除的记录
2. 点击 **"删除"** 按钮
3. 系统会弹出确认对话框，显示将要删除的记录ID
4. 确认后点击 **"是"** 完成删除
5. **删除操作不可恢复**，请谨慎操作

### 5. 搜索记录

在编辑之前，可以使用搜索功能快速定位：

1. 在搜索框中输入 **名称关键词**
2. 点击 **"搜索"** 按钮
3. 列表会筛选显示包含关键词的记录
4. 点击 **"显示全部"** 返回完整列表

## 编辑对话框特性

### 1. 滚动支持
- 对于字段较多的表（如品目信息），对话框支持垂直滚动
- 窗口大小：500x600像素
- 可通过鼠标滚轮或滚动条浏览所有字段

### 2. 自动类型转换
- 数字字段（价格、尺寸、汇率等）会自动转换为浮点数
- 空白输入自动转为 NULL
- 转换失败时保留原文本值

### 3. 字段验证
- **编码唯一性**：添加或修改时，系统检查编码是否重复
- **错误提示**：重复编码会显示具体错误信息
- **ID保护**：编辑时ID字段自动设为只读

### 4. 中文字段标签
- 所有字段都有对应的中文标签
- 未定义的字段直接显示英文字段名
- 便于理解和操作

## 数据库备份建议

由于编辑操作会直接修改数据库，建议定期备份：

```bash
# 手动备份数据库
copy base_data.db base_data_backup_20250106.db

# 或在程序外部使用 SQLite 工具备份
```

**自动备份**：可以在 `base_data_manager.py` 中添加自动备份功能。

## 常见问题

### Q1: 添加时提示"可能存在重复的编码"
**A**: 编码字段必须唯一。请检查是否已存在相同编码的记录，或使用搜索功能查找。

### Q2: 删除记录后能否恢复？
**A**: 不能。删除操作直接从数据库中移除记录，无法恢复。建议删除前确认，重要数据请先备份。

### Q3: 能否批量导入/导出数据？
**A**:
- **批量导入**：通过菜单 `基础数据` → `导入基础数据` 从Excel文件批量导入
- **批量导出**：目前不支持，可使用SQLite工具导出为CSV

### Q4: 修改数据后多久生效？
**A**: 立即生效。保存后数据直接写入SQLite数据库，刷新列表即可看到。

### Q5: 支持哪些数据类型？
**A**:
- 文本字段：存储为 TEXT
- 数字字段：存储为 REAL（浮点数）
- 空值：存储为 NULL

## 技术实现

### 数据库方法

`base_data_manager.py` 提供的核心方法：

```python
# 添加记录
mgr.add_record(table_name, data_dict)

# 更新记录
mgr.update_record(table_name, record_id, data_dict)

# 删除记录
mgr.delete_record(table_name, record_id)

# 查询单条记录
mgr.get_record_by_id(table_name, record_id)

# 获取表结构
mgr.get_table_columns(table_name)
```

### GUI实现

`亿看智能识别系统.py` 中的相关方法：

- `query_base_data()`: 查询窗口，包含增删改按钮
- `_edit_record_dialog()`: 编辑对话框，支持新增和编辑模式
- 字段映射字典：`field_labels` 提供中文标签

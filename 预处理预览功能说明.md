# 预处理预览功能说明

## 功能概述

在导入Excel文件并点击"开始转换并导出"后，如果启用了**摘要智能识别**功能，系统会在正式导出前显示一个**预处理预览窗口**，让您查看智能识别的处理结果。

## 预览窗口界面

### 窗口布局

预览窗口采用分页设计，包含两个标签页：

#### 1. 识别详情页

显示智能识别的详细信息：

- **顶部统计信息**：
  - 共处理多少行数据
  - 识别到多少行包含可识别信息

- **逐行显示识别结果**：
  - 每一行显示为独立的卡片
  - 显示Excel行号（便于定位）
  - 显示原始摘要内容
  - 显示识别到的所有字段（蓝色文本标识）

**示例显示**：
```
┌─ 第 2 行 ─────────────────────────────────────────┐
│ 摘要: 销售鞋类商品给ABEL CAPOTE，金额500美元        │
│ 识别结果:                                          │
│   • 摘要编码: 01                                    │
│   • 类型: 1                                         │
│   • 科目编码: 1122                                  │
│   • 往来单位编码: H0405                             │
│   • 往来单位名: ABEL CAPOTE                         │
│   • 金额: 500.0                                     │
└───────────────────────────────────────────────────┘
```

#### 2. 转换结果预览页

以表格形式显示最终的转换结果：

- 显示所有模板字段作为列
- 显示前20行转换后的数据
- 支持横向和纵向滚动查看
- 如果数据超过20行，会显示总行数提示

### 操作按钮

- **确认并导出**：确认结果无误，继续执行导出操作
- **取消**：取消本次转换，返回主界面（可以调整映射或修改数据后重试）

## 使用场景

### 场景1：检查识别准确性

在处理大批量数据前，先通过预览窗口检查几行样本数据：

1. 查看摘要是否被正确识别
2. 确认往来单位匹配是否准确
3. 检查金额、日期格式是否正确
4. 验证业务类型和科目编码是否符合预期

**如果发现问题**：
- 点击"取消"
- 调整摘要内容或手动映射字段
- 重新执行转换

### 场景2：验证混合映射结果

当同时使用手动映射和智能识别时：

1. 在"转换结果预览"页查看完整的数据
2. 确认手动映射的字段值正确
3. 确认智能识别补充的字段合理

### 场景3：数据审核

在导出前最后一次审核：

1. "识别详情"页：逐条查看识别逻辑
2. "转换结果预览"页：整体查看导出数据
3. 确认无误后点击"确认并导出"

## 显示规则

### 何时显示预览窗口

预览窗口**仅在以下条件都满足时**显示：

1. ✅ 勾选了"启用摘要智能识别（自动填充字段）"
2. ✅ 至少有一行数据被成功识别到信息

**不显示预览窗口的情况**：
- 未启用摘要智能识别 → 直接跳转到导出对话框
- 没有识别到任何信息 → 直接跳转到导出对话框

### 识别详情显示规则

- 只显示**有识别结果**的行
- 只显示**非空字段**（空值不显示）
- 按Excel原始行号排序

### 转换结果预览限制

- **最多显示前20行**（避免界面过于庞大）
- 如果超过20行，底部会提示总行数
- 实际导出时会包含所有行

## 窗口特性

### 滚动支持

- **识别详情页**：
  - 纵向滚动：查看多行识别结果
  - 横向滚动：查看长文本摘要

- **转换结果预览页**：
  - 纵向滚动：查看更多数据行
  - 横向滚动：查看更多字段列

### 模态窗口

预览窗口是**模态窗口**：

- 打开预览窗口时，主窗口被锁定
- 必须选择"确认并导出"或"取消"才能继续
- 防止在预览时误操作主界面

## 工作流程

```
[开始转换并导出]
        ↓
[执行字段映射和智能识别]
        ↓
[收集识别信息]
        ↓
    ┌──判断──┐
    │启用智能│
    │识别?   │
    └────────┘
      ↓ 是      ↓ 否
[显示预览窗口] [直接导出]
      ↓
   ┌──用户选择──┐
   │确认 | 取消 │
   └────────────┘
      ↓       ↓
   [导出]  [返回]
```

## 优势

### 1. 提高准确性

- 导出前可以发现识别错误
- 避免生成错误的凭证数据
- 减少后期修正工作

### 2. 增强信任度

- 透明展示识别过程
- 用户可以理解系统如何识别数据
- 增加对智能识别的信任

### 3. 学习优化

- 通过查看识别结果了解系统规则
- 发现规律后可以优化摘要格式
- 提高后续批次的识别准确率

### 4. 灵活控制

- 可以随时取消转换
- 不满意可以调整后重试
- 不会生成错误文件

## 注意事项

### 1. 预览数据量

- 识别详情页显示**所有**有识别结果的行
- 转换结果预览页只显示**前20行**
- 如果数据量很大（>100行），识别详情页加载可能需要几秒

### 2. 窗口大小

- 默认窗口大小：1200x700
- 可以拖拽调整窗口大小
- 建议在较大屏幕上使用以获得更好体验

### 3. 取消操作

- 点击"取消"不会保存任何数据
- 可以安全返回主界面重新操作
- 已识别的数据不会丢失（再次转换会重新识别）

### 4. 性能考虑

- 识别详情展示可能消耗较多内存（如果数据量大）
- 对于超大数据量（>1000行），建议分批处理
- 可以在主界面筛选后再导入

## 实用技巧

### 技巧1：先测试小样本

1. 从原始Excel中复制前10行数据到新文件
2. 导入小样本文件
3. 查看预览窗口的识别结果
4. 确认识别准确后再导入完整数据

### 技巧2：对比识别详情和转换结果

1. 在"识别详情"页找到某一行（如第5行）
2. 记住识别到的字段值
3. 切换到"转换结果预览"页
4. 在表格中找到对应行，验证数据是否一致

### 技巧3：发现识别规律

1. 查看多行识别详情
2. 总结哪些关键词触发了哪些业务类型
3. 了解往来单位匹配的模式
4. 在后续数据中使用相同格式提高识别率

### 技巧4：快速定位问题

如果某行识别结果不对：

1. 记下Excel行号（预览窗口显示）
2. 取消转换，返回主界面
3. 在原始Excel中找到该行
4. 手动映射该行的字段或修改摘要内容
5. 重新转换并验证

## 常见问题

### Q1: 为什么没有显示预览窗口？

**可能原因**：
1. 未勾选"启用摘要智能识别"
2. 没有任何数据被识别到
3. 摘要列未映射

**解决方案**：
- 确认勾选了智能识别选项
- 确认"摘要"字段已正确映射
- 检查摘要内容是否包含可识别关键词

### Q2: 预览窗口显示的数据和最终导出的不一致？

**不可能出现此情况**，因为：
- 预览窗口显示的就是即将导出的数据
- 点击"确认并导出"后不会再次处理数据
- 如发现不一致，可能是查看错行号

### Q3: 识别详情页加载很慢？

**原因**：数据量过大（>500行）

**解决方案**：
- 先查看"转换结果预览"页（只显示20行，加载快）
- 如果满意，直接点击"确认并导出"
- 或者分批处理数据

### Q4: 可以在预览窗口中修改数据吗？

**不可以**。预览窗口是**只读查看**。

如需修改：
1. 点击"取消"返回主界面
2. 调整列映射或修改原始Excel文件
3. 重新执行转换

### Q5: 转换结果预览只显示20行，其他数据会导出吗？

**会**。转换结果预览只是**样本显示**，实际导出会包含**所有行**。

预览限制20行是为了：
- 提高界面加载速度
- 避免界面过于庞大
- 提供快速的数据质量检查

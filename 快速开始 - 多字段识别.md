# 快速开始 - 多字段识别和默认值

## 功能亮点

系统现在支持：
- ✅ **多字段智能识别**：从日期、金额、汇率、摘要多个字段提取信息
- ✅ **默认值设置**：统一设置部门、外币金额、汇率等默认值
- ✅ **智能优先级**：手动映射 > 原始数据 > 智能识别 > 默认值


## 实际应用场景

### 场景1：有完整字段的Excel

**Excel格式**：
```
| 日期       | 客户名称  | 金额   | 汇率 | 摘要        |
|-----------|---------|-------|------|------------|
| 2025-01-15| ABEL    | 500   | 6.5  | 销售鞋类    |
```

**操作**：
1. 映射字段：
   - 凭证日期 ← 日期
   - 往来单位名 ← 客户名称
   - 金额 ← 金额
   - 汇率 ← 汇率
   - 摘要 ← 摘要
2. 设置默认值：部门=10001、外币金额=0
3. 启用智能识别
4. 转换

**效果**：
- 凭证日期、金额、汇率、往来单位名：从手动映射
- 摘要编码、类型、科目编码：从摘要智能识别
- 部门、外币金额：从默认值

### 场景2：字段不完整的Excel

**Excel格式**：
```
| Date       | Description              | amt   |
|-----------|--------------------------|-------|
| 2025-01-15| 采购原材料入库            | 1000  |
```

**操作**：
1. 启用智能识别（会自动识别Date→凭证日期、amt→金额）
2. 映射：摘要 ← Description
3. 设置默认值：部门=10001、汇率=6.5、外币金额=0
4. 转换

**效果**：
- 凭证日期：从Date字段自动识别
- 金额：从amt字段自动识别
- 摘要编码、类型、科目编码：从Description智能识别
- 汇率、部门、外币金额：从默认值

### 场景3：只有摘要的Excel

**Excel格式**：
```
| 摘要                                      |
|------------------------------------------|
| 2025-01-15收到ABEL货款500美元，汇率6.5    |
```

**操作**：
1. 映射：摘要 ← 摘要
2. 设置默认值：部门=10001、外币金额=0
3. 启用智能识别
4. 转换

**效果**：
- 凭证日期：从摘要识别 `20250115`
- 金额：从摘要识别 `500`
- 汇率：从摘要识别 `6.5`
- 往来单位：从摘要识别 `ABEL`
- 业务类型：从摘要识别（收款）
- 部门、外币金额：从默认值

  - 来源：默认值
  - 说明：未手动映射，未从数据提取，使用默认值

  - 来源：默认值

```
部门: 10001

汇率: 6.5
```

适用于：大部分是巴拿马部门的外币交易

### 配置2：中国本币业务

```
部门: 10002
外币金额: 0
汇率: 1
```

适用于：大部分是中国部门的本币交易

### 配置3：混合业务

```
外币金额: 0
汇率: 1
```

不设置部门，让系统从摘要识别或手动映射

## 技巧总结

### 1. 分批导入不同业务

- 导入巴拿马数据前：设置部门=10001、汇率=6.5
- 导入中国数据前：设置部门=10002、汇率=1
- 避免混合导入需要后期修改

### 2. 利用字段名变体

系统支持：
- 日期：date、Date、日期、凭证日期
- 金额：amount、Amount、amt、金额
- 汇率：rate、Rate、exchange_rate、汇率

Excel列名使用这些标准名称，无需手动映射

### 3. 结合手动映射

- 确定的字段：手动映射（优先级最高）
- 不确定的字段：智能识别
- 固定的字段：默认值

### 4. 使用预览验证

- 每次转换前查看预处理预览
- 确认默认值是否正确应用
- 确认优先级是否符合预期

## 常见问题

### Q: 默认值没有生效？
**A**: 检查是否该字段已手动映射或已从数据识别到值（优先级更高）

### Q: 想强制使用默认值？
**A**: 取消该字段的手动映射，并确保原始数据中没有该字段

### Q: 如何清空默认值？
**A**: 打开默认值对话框，点击"清空"按钮

### Q: 默认值会保存吗？
**A**: 不会持久化，关闭程序后清空，下次需重新设置

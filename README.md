# 亿看智能识别系统

一般凭证 Excel 格式转换工具，集成基础数据管理功能。

## 功能概述

### 1. Excel 格式转换
- 自动识别和匹配列标题
- 智能字段映射（支持同义词识别）
- 字段格式转换（日期、数字、文本）
- 保留模板格式和批注
- **摘要智能识别** - 从摘要内容自动识别并填充多个字段
- **预处理预览** - 导出前查看识别结果和转换数据

### 2. 基础数据管理
- 自动导入和管理7类基础数据
- 数据库持久化存储
- 快速查询和搜索功能
- 增删改查完整功能

### 3. 经营分析报告（升级版）
- 自动补齐图表与图表说明（按表生成趋势/分布图）
- 现金流量表（估算）、预算执行与偏差、异常预警、预测与滚动预算
- 产品/品类贡献毛利、客户贡献与回款、渠道贡献、存货健康度与滞销清单
- 费用结构与弹性、多主体汇总（如有多目录）、币种汇总（如有币种字段）
- 数据质量检查与审计日志

## 安装依赖

```bash
pip install -r requirements.txt
```

可选功能（图片识别/本地AI）依赖：

```bash
pip install -r requirements-optional.txt
```

提示：如需避免在本地保存 API Key，可设置环境变量 `YIKAN_AI_API_KEY` 或 `ZHIPU_API_KEY`，程序会自动读取。

## 使用方法

### 启动程序

```bash
python 亿看智能识别系统.py
```

### 基础数据管理

程序启动时会自动检查并加载基础数据：
- 如果 `base_data.db` 不存在或为空，会自动从 `基础数据/基础数据/` 目录导入所有Excel文件
- 导入成功后，数据保存在 SQLite 数据库中，下次启动无需重新导入

#### 手动导入基础数据

通过菜单：`基础数据` -> `导入基础数据`

支持的数据类型：
- 币种 (currency) - 1条记录
- 部门 (department) - 2条记录
- 仓库 (warehouse) - 3条记录
- 科目编码 (account_subject) - 254条记录
- 品目信息 (product) - 1674条记录
- 往来单位 (business_partner) - 480条记录
- 账户 (bank_account) - 21条记录

总计约 2435 条基础数据记录

#### 导出模板 / 批量导入

- `导出基础数据模板`：一键生成标准模板Excel（首行含说明，第二行为表头）
- `批量导入基础数据Excel`：选择目录自动匹配文件名并导入

#### 查询基础数据

通过菜单访问：
- `基础数据` -> `查看统计信息` - 显示各表记录数
- `基础数据` -> `查询科目编码` - 查询会计科目
- `基础数据` -> `查询往来单位` - 查询客户供应商
- `基础数据` -> `查询品目信息` - 查询产品信息
- `基础数据` -> `查询部门` - 查询组织部门
- `基础数据` -> `查询账户` - 查询银行账户

查询窗口支持：
- 按名称关键词搜索
- 显示全部记录
- 树形视图展示结果
- **新增记录** - 点击"新增"按钮添加新数据
- **编辑记录** - 选中记录后点击"编辑"按钮修改数据
- **删除记录** - 选中记录后点击"删除"按钮删除数据

#### 编辑基础数据

每个查询窗口都包含完整的增删改功能：

1. **新增记录**
   - 点击"新增"按钮
   - 在弹出的对话框中填写各字段
   - 点击"保存"完成添加
   - 编码字段(code)必须唯一

2. **编辑记录**
   - 在列表中选中要编辑的记录
   - 点击"编辑"按钮
   - 修改字段内容
   - 点击"保存"完成更新
   - ID字段不可编辑

3. **删除记录**
   - 在列表中选中要删除的记录
   - 点击"删除"按钮
   - 确认删除操作
   - 记录将从数据库中永久删除

### Excel 转换功能

1. **选择原始 Excel 文件**
   - 点击"浏览..."按钮选择文件
   - 选择要处理的工作表

2. **启用智能识别（可选）**
   - 勾选"启用摘要智能识别（自动填充字段）"复选框（默认启用）
   - **多字段智能识别**：系统会从以下字段自动提取信息
     - **日期字段** → 凭证日期（支持：日期、date、Date等）
     - **金额字段** → 金额（支持：金额、amount、amt等）
     - **汇率字段** → 汇率（支持：汇率、rate等）
     - **摘要字段** → 多个字段：
       - 业务类型（销售、采购、收款、付款等）
       - 往来单位名称和编码
       - 科目编码
       - 部门信息
       - 金额、日期（补充识别）
   - 详细说明请查看 `摘要智能识别说明.md` 和 `多字段识别和默认值说明.md`

3. **设置默认值（可选）**
   - 点击"设置默认值..."按钮
   - 为常用字段设置默认值：
     - 部门（例如：10001）
     - 外币金额（例如：0）
     - 汇率（例如：1 或 6.5）
     - 序号、会计凭证No.等
   - 默认值在字段未映射且未识别时自动应用
   - 详细说明请查看 `多字段识别和默认值说明.md`

4. **自动识别列匹配**
   - 点击"自动识别匹配"按钮
   - 系统会根据列名相似度自动匹配
   - 阈值：0.6（可在代码中调整）

5. **手动调整映射**
   - 在下拉框中调整不准确的映射
   - 可选择"<留空>"跳过某些列
   - 注：**字段值优先级**：手动映射 > 原始数据字段 > 智能识别 > 默认值

6. **查看字段说明**
   - 点击"查看"按钮查看模板字段的批注说明

7. **预处理预览与导出**
       - 点击"开始转换并导出"
      - 如启用了智能识别且有识别结果，会弹出预处理预览窗口：
        - **识别详情页**：显示每行的摘要和智能识别结果
        - **转换结果预览页**：显示最终转换后的数据表格（前20行）
      - 在预览窗口中可以：
        - 查看识别准确性
        - 审核转换结果
        - 点击"确认并导出"继续或"取消"返回调整
      - 选择保存位置
      - 生成符合模板格式的新Excel文件
      - 详细说明请查看 `预处理预览功能说明.md`
   
   8. **智能还原编码**
      - 点击"智能还原编码"按钮
      - 系统会自动尝试修复"往来单位编码/名"和"品目编码/名"列中的错误编码
      - 修复依据：基础数据中的名称、同义词（Match Items）以及相似度模糊匹配（阈值0.85）
      - 修复完成后会弹出统计报告，显示修复数量

## 字段转换规则

### 日期字段
- 支持格式：YYYYMMDD、YYYY-MM-DD、YYYY/MM/DD、YYYY.MM.DD
- 支持 Excel 序列号
- 空值默认为当天日期

### 数字字段
- 自动去除千位分隔符
- 按配置的整数位/小数位截断
- 空值转换为空字符串

### 文本字段
- 按配置的最大长度截断
- 空值转换为空字符串

## 模板字段配置

在 `亿看智能识别系统.py` 中修改 `FIELD_RULES` 字典来自定义字段规则：

```python
FIELD_RULES = {
    "凭证日期": {"type": "date"},
    "序号": {"type": "text", "max_len": 4},
    "金额": {"type": "number", "max_int_len": 15, "max_decimal_len": 2},
    # ... 更多字段
}
```

## 同义词配置

在 `FIELD_SYNONYMS` 字典中添加字段别名，提高自动匹配准确率：

```python
FIELD_SYNONYMS = {
    "科目编码": ["科目代码", "科目编号", "会计科目", "科目"],
    "往来单位名": ["往来单位名称", "客户名称", "供应商名称", "单位名称"],
    # ... 更多同义词
}
```

## 注意事项

1. **Template.xlsx 必须存在**：程序启动前确保模板文件在同一目录
2. **基础数据目录**：确保 `基础数据/基础数据/` 目录存在且包含所需Excel文件
3. **Excel文件格式**：基础数据文件第一行为公司名称，第二行为表头
4. **数据库文件**：`base_data.db` 会自动创建，建议定期备份
5. **编码问题**：Windows命令行可能显示乱码，不影响程序功能

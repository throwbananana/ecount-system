## 标签页 1: Excel凭证转换

### 功能说明

将任意格式的Excel文件转换为标准的一般凭证模板格式。

### 界面布局

```
┌────────────────────────────────────────────────┐
│ 原始 Excel: [_____________________] [浏览...]  │
│ 工作表:     [▼ 下拉选择]                        │
├────────────────────────────────────────────────┤
│ ┌──字段映射（模板列 -> 原始列）──────────────┐ │
│ │ 凭证日期   [▼ 原始列选择]     [查看批注]   │ │
│ │ 序号       [▼ 原始列选择]     [查看批注]   │ │
│ │ ...                                        │ │
│ └────────────────────────────────────────────┘ │
├────────────────────────────────────────────────┤
│ [自动识别匹配] [预览映射] [智能还原编码] [转换导出] │
└────────────────────────────────────────────────┘
```

### 使用步骤

1. **选择文件**: 点击"浏览..."选择要转换的Excel文件
2. **选择工作表**: 从下拉框选择要处理的工作表
3. **自动匹配**: 点击"自动识别匹配"，系统自动匹配列
4. **手动调整**: 在下拉框中调整不正确的映射
5. **智能还原**: (可选) 点击"智能还原编码"，系统自动利用模糊匹配修复往来单位和品目编码
6. **开始转换**: 点击"开始转换并导出"，选择保存位置

### 功能特点

- ✅ 智能列名匹配（支持同义词识别）
- ✅ **智能编码还原**（支持模糊匹配修复）
- ✅ 可手动调整映射
- ✅ 查看模板字段说明
- ✅ 保留模板格式和批注

## 标签页 2: 基础数据管理

### 功能说明

在主窗口中直接查看和编辑7类基础数据，无需打开新窗口。

### 界面布局

```
┌──────────────┬────────────────────────────────────────┐
│ 数据类型     │ 搜索: [______] [搜索] [显示全部]        │
│              │ [删除] [编辑] [新增] ← 操作按钮        │
│ ○ 币种       ├────────────────────────────────────────┤
│ ● 部门       │ ┌───────────────────────────────────┐ │
│ ○ 仓库       │ │ id │ code  │ name    │ is_active│ │
│ ○ 科目编码   │ ├────┼───────┼─────────┼──────────┤ │
│ ○ 品目信息   │ │ 1  │10001  │巴拿马... │ 是      │ │
│ ○ 往来单位   │ │ 2  │10002  │巴拿马... │ 是      │ │
│ ○ 账户       │ └───────────────────────────────────┘ │
│              ├────────────────────────────────────────┤
│              │ 部门 - 共 2 条记录                      │
└──────────────┴────────────────────────────────────────┘
```

### 使用步骤

#### 1. 查看数据

- **切换数据类型**: 点击左侧单选按钮（币种、部门、仓库等）
- **自动加载**: 切换后右侧表格自动显示该类型的所有数据
- **滚动查看**: 使用滚动条查看更多记录

#### 2. 搜索数据

- 在搜索框输入**名称关键词**
- 点击"搜索"按钮
- 点击"显示全部"返回完整列表

#### 3. 新增记录

1. 点击右上角的 **"新增"** 按钮
2. 在弹出的对话框中填写字段
3. 必填字段：
   - **编码 (code)** - 必须唯一
   - **名称 (name)**
4. 点击"保存"完成添加

#### 4. 编辑记录

**方法一：双击编辑**
- 双击表格中的任意记录
- 弹出编辑对话框
- 修改字段后点击"保存"

**方法二：按钮编辑**
- 单击选中要编辑的记录
- 点击"编辑"按钮
- 修改字段后点击"保存"

#### 5. 删除记录

1. 单击选中要删除的记录
2. 点击"删除"按钮
3. 确认删除操作
4. **注意**: 删除不可恢复！

### 功能特点

- ✅ 一体化界面，无需打开新窗口
- ✅ 左右分栏布局，操作流畅
- ✅ 双击快速编辑
- ✅ 实时搜索和过滤
- ✅ 状态栏显示记录数
- ✅ 支持所有7类基础数据

## 新旧界面对比

### 旧版界面

```
主窗口: Excel转换功能
  └── 菜单 → 基础数据 → 查询XXX → 打开新窗口
        └── 新窗口中查看和编辑
```

**缺点**：
- ❌ 需要打开多个窗口
- ❌ 窗口管理麻烦
- ❌ Excel转换和数据管理分离

### 新版界面

```
主窗口: 标签页式界面
  ├── 标签1: Excel转换
  └── 标签2: 基础数据管理（集成查看和编辑）
```

**优点**：
- ✅ 单窗口操作
- ✅ 标签页切换方便
- ✅ 两大功能集成
- ✅ 界面简洁统一

## 常见问题

### Q1: 如何切换查看不同类型的数据？
**A**: 点击左侧的单选按钮（币种、部门、仓库等），右侧表格会自动切换。

### Q2: 能否同时编辑多条记录？
**A**: 不支持批量编辑，需要逐条编辑。可以使用"导入基础数据"功能批量更新。

### Q3: 编辑对话框的字段太多，怎么查看？
**A**: 对话框支持滚动，使用鼠标滚轮或右侧滚动条查看所有字段。

### Q4: 新旧界面有什么区别？
**A**: 新界面将基础数据管理集成到主窗口的第二个标签页中，无需打开新窗口。旧版菜单功能仍然保留。

### Q5: 数据保存在哪里？
**A**: 所有基础数据保存在 `base_data.db` SQLite数据库文件中。

## 技术说明

### 界面框架

- **Tkinter**: Python标准GUI库
- **ttk.Notebook**: 标签页控件
- **ttk.Treeview**: 数据表格控件

### 布局方式

- **Pack布局**: 主要使用pack管理器
- **Grid布局**: 部分使用grid（Excel转换标签页）
- **响应式**: 窗口缩放时自动调整

### 数据绑定

- **StringVar**: 搜索框、当前表名
- **实时更新**: 切换数据类型立即加载
- **事件绑定**: 双击编辑、单选按钮切换
